Simple URL shortner

By Slaven Robić

Demonstrating simple URL shorting

API

ENDPOINT: /Account, example: http://localhost:49179/Account, body: {"accountId":"dd1s"}, returns JSON: {"success":true,"password":"VlDPIP79","description":"Your account is opened."} or {"success": false,"description": "User name 'dd1s' is already taken."}


ENDPOINT: /Register, example: http://localhost:49179/Register, Header: Authorization password (password is auth token), body: {"RedirectURL":"www.youtube.com", "RedirectType":301}, returns JSON: {"shortUrl":"http://localhost:49179/c"} or {"error":"URL Verification Error"} or {Unathorized}


ENDPOINT: /Statistic/{AccountId}, example: http://localhost:49179/statistic/aaaa, Header: Authorization password (password is auth token), body: {"RedirectURL":"www.youtube.com", "RedirectType":301}, returns JSON array: [{"redirectURL":"http://www.youtube.com","hitCount":2},{"redirectURL":"http://www.nba.com","hitCount":2}] or JSON {"error":"You can not see stats of another user!"} or {Unathorized}


WEB APP

Web app has Register option which needs AccountId. On user screen we can register new links and see stats. There is this help page, and Logout button.
